# model/

Python/MATLAB reference model and stability study for the 2D FDTD mesh.

Planned contents:
- `fdtd_ref.py` — pure-Python reference implementation of the node update equation
- `stability_study.py` — CFL sweep; plots `gamma^2` vs blow-up threshold
- `impulse_response.py` — generates reference impulse-response data for RTL comparison
